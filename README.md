# Heart Disease Prediction Using Machine Learning

## Project Overview
This project predicts the presence of heart disease using machine learning techniques.

## Dataset
Heart Disease UCI Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Models Used
- Logistic Regression
- Random Forest Classifier

## Results
- Logistic Regression Accuracy: 79.51%
- Random Forest Accuracy: 98.54%

## Conclusion
Random Forest outperformed Logistic Regression and was selected as the final model.
